Sttxt - SMS to txt

Basic app to save received SMS messages to txt file for MRE platform (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. SMS data saved to "Sttxt.txt" file (changing app name change default text file name) on storage card or phone memory. LEFT_SOFTKEY - start saving all SMS to single "Sttxt.txt" file , KEY_NUM1 - start saving all SMS to different files for each SMS, RIGHT_SOFTKEY - exit app. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top/ https://vxpatch.luxferre.top. Application file - "Sttxt.vxp".
